### Level 2 Checklist (Optional/Low Flexibility)

- [ ] **Use [WebP format](https://developers.google.com/speed/webp)** for images instead of JPEG/PNG to reduce file size.
- [ ] **Ensure that your site is using HTTPS** for security and SEO benefits.
- [ ] **Optimize JavaScript files** to reduce loading time by minifying them.
- [ ] **Use [Browser Caching](https://developers.google.com/web/fundamentals/performance/optimizing-content-efficiency/http-cache)** for static resources to enhance loading speeds.
- [ ] **Look into [HTTP/2](https://http2.github.io/)** if your server supports it for improved performance.
- [ ] **Implement [Lazy Loading](https://web.dev/lazy-loading-images/)** for images and videos to improve initial load times.
